Description :
Steganography : The Practice of hiding information within other data, such as images, audio, or text, to conceal messages,
                This project Based on Image Steganography (.bmp)
                Hiding messages inside image files by altering pixel values.
                The Least Significant Bit (LSB) method is the most common, where the smallest pixel value changes encode data without 			noticeable visual differences.
